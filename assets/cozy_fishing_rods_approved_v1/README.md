# Cozy Fishing Rods Approved v1

Tres canas aprobadas en estilo low-poly cute para el set cozy:

- `fairy_rod/`: cana pastel magica con alas, carrete decorativo y charm de flor.
- `wizard_rod/`: cana azul/violeta con luna creciente, detalles dorados y charm celestial.
- `villager_rod/`: cana rustica de madera con hoja, carrete natural y farol colgante.

Cada asset incluye:

- `model.glb`: modelo listo para runtime.
- `source.blend`: fuente editable en Blender.
- `perspective.png`: preview renderizado.

Los scripts generadores estan en `../../scripts/create_*_rod_blender.py`.
